## 한눈 요약

- **세 개의 게임**: 로그라이크 카드 전투 **삼국 영웅전**, 보드 배치 카드 게임 **히어로즈 블러드**, 레이드 보스전 **삼국 대장전**(영웅전 덱 공유).
- **순수 웹**: HTML + CSS + Vanilla JS, 빌드/의존성 0, 오프라인 동작, 모바일 지원.
- **게임 간 연동**: 히어로즈 블러드 결과(승/패 포인트)는 영웅전 골드로 정산되고 **3연승 시 제갈량** 획득, **대장전 보스 격파**로 전용 장수를 획득해 영웅전에 합류합니다.
- **검증**: 브라우저 없이 두 게임을 끝까지 자동 플레이하는 헤드리스 하니스와 AI 자가대국 밸런싱 스크립트로 회귀를 잡습니다.
- **배포**: GitHub Pages (`gnsyoo.github.io/tcg-game-mockup/`).
