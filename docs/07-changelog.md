# 07 · 변경 이력 (주요 마일스톤)

시간순 요약입니다. 세부 규칙은 각 게임 문서를 참고하세요.

## 초기

- 두 게임(영웅전 / 히어로즈 블러드) + 첫 페이지(난이도 상/중/하) 웹 목업.
- HTML+CSS+Vanilla JS, 의존성 0, 모바일 지원. GitHub Pages 배포.

## 아트 · 테마

- 사이트 제목 **「삼국지 카드 게임」**.
- 전 카드/적/보스를 **삼국지 장수**로 리스킨. **절차적 초상화** + CSS 아트 + 효과음 + 전투 연출 강화.
- 퀸즈 블러드 → **히어로즈 블러드**로 명칭 변경.

## 영웅전 — 캠페인/전투 구조

- 2단계 캠페인: **8개 역사 전역 × 각 12회 출진**, 마지막 출진은 **적장(보스)전**, 적장 격파 시 다음 전역 해금.
- 카드 전투 **3구역화**: 왼쪽 뽑을 더미 / 가운데 공격·방어 덱(3장) / 오른쪽 사용한 더미.
  턴 종료 시 남은 가운데 카드가 **방어**로 전환, 뽑을 더미가 비면 사용한 더미를 셔플 보충.
- 파티(=카드 덱) 최대 **30명**.

## 히어로즈 블러드 — 규칙

- 정산: **배치한 모든 카드의 무력 총합**이 높은 쪽 승리.
- 종료: **상대 패스가 아니라, 내가 패스를 2번** 하면 종료 여부를 선택.

## 영웅전 — 주공/무기/보물 개편

- **주공(나)** 도입: 시작 **HP 200 · MP 50**. 적은 주공 HP를 공격, 주공 전사 시 패배.
  HP는 모험 내내 유지(승리 시 일부 회복, 전역 평정 시 완전 회복). MP는 매 전투 충전, 스킬당 3~5 소모.
- **무기 장착**(장수당 1개): 공격력+/2회 연속/독/주공 HP·MP 증가 등 9종.
  **출진 5·10회 뒤 보물상자**, 저잣거리에서도 구매.
- 적 **독** 누적(턴마다 피해). 단일 주공 모델에 맞춰 난이도 재밸런싱.

## 이번 업데이트 (현재)

- **엔딩 크레딧**: 천하통일(사마염 격파) 후 평정 전역·장수 명단·유물·난이도를 롤업하는 크레딧 화면.
- **장수 중복 수집 방지**: 이미 보유한 장수는 보상/상점에 제시되지 않음(모두 보유 시 보물로 대체).
- **장수 12종 추가**: 가후 · 서황 · 초선 · 소교 · 대교 · 악진 · 만총 · 장료 · 조비 · 공손찬 · 위연 · 하진 → **총 50종**.
  - (요청의 "악덕"은 위 오자양장 테마에 맞춰 **악진(Yue Jin)** 으로 반영.)
- **매혹 스킬**: 초선·소교·대교는 적 1명을 **1턴 매혹**(행동 불가)시킴.
- **사마의 OP화**: 「화계」 전체 14 피해(총사령관급).
- **SSR 등급 신설**: 사마의·제갈량·관우·조운·하후돈·전위·조조를 **SSR**로 승급(능력치 상향).
  **SSR은 무기 2개 장착** 가능.
- **두 게임 골드 연동**: 히어로즈 블러드 승리 시 포인트만큼 영웅전 골드 **추가**, 패배 시 포인트만큼 **차감**.
- **문서화**: `docs/`에 기능 문서 분리 정리(이 폴더).

## UI/UX 개선 (현재)

삼국 영웅전
- 전투 가운데(공격·방어) 카드 영역을 내부 래퍼로 감싸 **가로 스크롤** 처리 — 카드 4장↑일 때 좌측 끝 카드가 잘리지 않음.
- 지도 화면의 상시 파티 나열을 **👥 장수 / 🗡️ 장비 버튼 + 팝업**으로 정리.
  장비 팝업은 보유 무기와 착용 중인 장수/미장착 수량을 보여줌.

히어로즈 블러드
- 진입 시 **덱 구성 화면부터** 표시(덱 15장 + 판 크기 선택 후 시작).
- **판 크기 3×5 · 4×5 · 5×5** 선택 추가(`ROWS` 가변화, `qb_rows` 저장).
- 종료 정산(영웅전 골드) 메시지를 **정돈된 배지**(`#endGold`)로 재구성.

## 전투 시스템 확장 (현재)

삼국 영웅전
- **치명타(크리티컬)**: 기본 공격이 기본 1% 확률로 **공격력 2배**. 적도 1% 확률로 치명타.
- **치명타 무기 3종 추가**: 청강검(+5%)·의천검(+7%)·자웅일대검(+10%) → 무기 **총 12종**.
- **무기 장착 수 등급화**: **C=0(장착 불가) · R=1 · SR=2 · SSR=3**.
- **MP 소모전화**: MP도 HP처럼 모험 내내 유지. **전투 시작 시 10% 회복**, 전역 평정 시 완전 회복,
  **회복 스킬은 HP 회복량의 1/4만큼 MP도 회복**. 이에 맞춰 적 수치를 하향 재밸런싱(봇 ~98/50/10%).

공통
- 상단 메뉴를 **우측 상단 플로팅 버튼(펼침형)**으로 변경, 패널 정보는 줄바꿈 없이 정렬.
- 히어로즈 블러드 **덱 구성 카드에 강화(영역) 패턴 표시** 추가.

## UI/규칙 다듬기 (현재)

삼국 영웅전
- 지도의 **출진/저잣거리 버튼 가운데 정렬**, 그 아래 **주공 HP·MP·골드 상태** 표시.
- 회복 스킬 MP가 줄던 버그 수정 → **순증가**(소모 없음, round(HP/4)).
- 메뉴에 **📖 게임 가이드** 모달 추가(가독성 있는 플레이 안내).
- 회복 스킬 버프로 약간 쉬워져 봇 클리어율 ~100/60/18%.

히어로즈 블러드
- 첫 화면(랜딩) **설명 업데이트**(판 크기·44종·골드 정산).
- 상대/당신 옆 **덱·패 글자 제거**(아이콘 🃏/✋로 대체).
- **배치 규칙 변경**: 칸의 폰 레벨 **이상**의 등급 카드만 배치 가능(`def.rank >= cell.rank`).
  예) 레벨 2 칸 → 등급 2·3·4·5 카드. 규칙 모달/랜딩/문서 갱신.

## 컬렉션·정리 (현재)

- 랜딩 페이지에서 부제/푸터 문구 제거(타이틀만 강조).
- 히어로즈 블러드 **판 크기 5×5 제거**(3×5·4×5만 유지).
- 삼국 영웅전 메뉴에 **📕 장수 컬렉션 / 🗡️ 장비 컬렉션(도감)** 추가 —
  전체 카탈로그(장수 50·무기 12) 중 **수집한 것은 밝게, 미수집은 어둡게(🔒)** 표시.
  수집 현황은 모험과 무관하게 영구 보존(`hw_collected_heroes`, `hw_collected_weapons`).

## 신규 모드·콘텐츠 (현재)

- **명칭 변경**: 삼국지 영웅모집 → **삼국 영웅전**.
- **장비 25종**: 기존 12종 + 13종 추가(공격·2회 연속·독·치명타·주공 HP/MP 등 다양한 조합).
- **제갈량 전용화**: 히어로즈 블러드 **3연승** 시에만 획득(`exclusive:'qb'`), 일반 보상/상점에서 제외.
- **삼국 대장전(레이드) 신규 게임**: 영웅전 **덱 공유**, 8개 적장이 거대 HP(약 12배)로 등장.
  보스 격파 시 **전용 SSR 장수 8종**(맹장 화웅 등) 획득 — 대장전에서만 획득 가능(`exclusive:'raid'`).
- **게임 간 장수 합류**: QB 연승·대장전 보상 장수는 `hw_grant_heroes`로 대기 → 영웅전 진입 시 합류.
- 장수 총 **58종**(일반 50 + 대장전 전용 8), 랜딩에 대장전 카드 추가. 제갈량 전용화로 hard 난이도 소폭 하향 재조정.
