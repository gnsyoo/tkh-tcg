# 07 · 변경 이력 (주요 마일스톤)

시간순 요약입니다. 세부 규칙은 각 게임 문서를 참고하세요.

## 초기

- 두 게임(영웅모집 / 히어로즈 블러드) + 첫 페이지(난이도 상/중/하) 웹 목업.
- HTML+CSS+Vanilla JS, 의존성 0, 모바일 지원. GitHub Pages 배포.

## 아트 · 테마

- 사이트 제목 **「삼국지 카드 게임」**.
- 전 카드/적/보스를 **삼국지 장수**로 리스킨. **절차적 초상화** + CSS 아트 + 효과음 + 전투 연출 강화.
- 퀸즈 블러드 → **히어로즈 블러드**로 명칭 변경.

## 영웅모집 — 캠페인/전투 구조

- 2단계 캠페인: **8개 역사 전역 × 각 12회 출진**, 마지막 출진은 **적장(보스)전**, 적장 격파 시 다음 전역 해금.
- 카드 전투 **3구역화**: 왼쪽 뽑을 더미 / 가운데 공격·방어 덱(3장) / 오른쪽 사용한 더미.
  턴 종료 시 남은 가운데 카드가 **방어**로 전환, 뽑을 더미가 비면 사용한 더미를 셔플 보충.
- 파티(=카드 덱) 최대 **30명**.

## 히어로즈 블러드 — 규칙

- 정산: **배치한 모든 카드의 무력 총합**이 높은 쪽 승리.
- 종료: **상대 패스가 아니라, 내가 패스를 2번** 하면 종료 여부를 선택.

## 영웅모집 — 주공/무기/보물 개편

- **주공(나)** 도입: 시작 **HP 200 · MP 50**. 적은 주공 HP를 공격, 주공 전사 시 패배.
  HP는 모험 내내 유지(승리 시 일부 회복, 전역 평정 시 완전 회복). MP는 매 전투 충전, 스킬당 3~5 소모.
- **무기 장착**(장수당 1개): 공격력+/2회 연속/독/주공 HP·MP 증가 등 9종.
  **출진 5·10회 뒤 보물상자**, 저잣거리에서도 구매.
- 적 **독** 누적(턴마다 피해). 단일 주공 모델에 맞춰 난이도 재밸런싱.

## 이번 업데이트 (현재)

- **엔딩 크레딧**: 천하통일(사마염 격파) 후 평정 전역·장수 명단·유물·난이도를 롤업하는 크레딧 화면.
- **장수 중복 수집 방지**: 이미 보유한 장수는 보상/상점에 제시되지 않음(모두 보유 시 보물로 대체).
- **장수 12종 추가**: 가후 · 서황 · 초선 · 소교 · 대교 · 악진 · 만총 · 장료 · 조비 · 공손찬 · 위연 · 하진 → **총 50종**.
  - (요청의 "악덕"은 위 오자양장 테마에 맞춰 **악진(Yue Jin)** 으로 반영.)
- **매혹 스킬**: 초선·소교·대교는 적 1명을 **1턴 매혹**(행동 불가)시킴.
- **사마의 OP화**: 「화계」 전체 14 피해(총사령관급).
- **SSR 등급 신설**: 사마의·제갈량·관우·조운·하후돈·전위·조조를 **SSR**로 승급(능력치 상향).
  **SSR은 무기 2개 장착** 가능.
- **두 게임 골드 연동**: 히어로즈 블러드 승리 시 포인트만큼 영웅모집 골드 **추가**, 패배 시 포인트만큼 **차감**.
- **문서화**: `docs/`에 기능 문서 분리 정리(이 폴더).
