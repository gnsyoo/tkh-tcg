# 07 · 변경 이력 (주요 마일스톤)

시간순 요약입니다. 세부 규칙은 각 게임 문서를 참고하세요.

## 초기

- 두 게임(영웅전 / 히어로즈 블러드) + 첫 페이지(난이도 상/중/하) 웹 목업.
- HTML+CSS+Vanilla JS, 의존성 0, 모바일 지원. GitHub Pages 배포.

## 아트 · 테마

- 사이트 제목 **「삼국지 카드 게임」**.
- 전 카드/적/보스를 **삼국지 장수**로 리스킨. **절차적 초상화** + CSS 아트 + 효과음 + 전투 연출 강화.
- 퀸즈 블러드 → **히어로즈 블러드**로 명칭 변경.

## 영웅전 — 캠페인/전투 구조

- 2단계 캠페인: **8개 역사 전역 × 각 12회 출진**, 마지막 출진은 **적장(보스)전**, 적장 격파 시 다음 전역 해금.
- 카드 전투 **3구역화**: 왼쪽 뽑을 더미 / 가운데 공격·방어 덱(3장) / 오른쪽 사용한 더미.
  턴 종료 시 남은 가운데 카드가 **방어**로 전환, 뽑을 더미가 비면 사용한 더미를 셔플 보충.
- 파티(=카드 덱) 최대 **30명**.

## 히어로즈 블러드 — 규칙

- 정산: **배치한 모든 카드의 무력 총합**이 높은 쪽 승리.
- 종료: **상대 패스가 아니라, 내가 패스를 2번** 하면 종료 여부를 선택.

## 영웅전 — 주공/무기/보물 개편

- **주공(나)** 도입: 시작 **HP 200 · MP 50**. 적은 주공 HP를 공격, 주공 전사 시 패배.
  HP는 모험 내내 유지(승리 시 일부 회복, 전역 평정 시 완전 회복). MP는 매 전투 충전, 스킬당 3~5 소모.
- **무기 장착**(장수당 1개): 공격력+/2회 연속/독/주공 HP·MP 증가 등 9종.
  **출진 5·10회 뒤 보물상자**, 저잣거리에서도 구매.
- 적 **독** 누적(턴마다 피해). 단일 주공 모델에 맞춰 난이도 재밸런싱.

## 이번 업데이트 (현재)

- **엔딩 크레딧**: 천하통일(사마염 격파) 후 평정 전역·장수 명단·유물·난이도를 롤업하는 크레딧 화면.
- **장수 중복 수집 방지**: 이미 보유한 장수는 보상/상점에 제시되지 않음(모두 보유 시 보물로 대체).
- **장수 12종 추가**: 가후 · 서황 · 초선 · 소교 · 대교 · 악진 · 만총 · 장료 · 조비 · 공손찬 · 위연 · 하진 → **총 50종**.
  - (요청의 "악덕"은 위 오자양장 테마에 맞춰 **악진(Yue Jin)** 으로 반영.)
- **매혹 스킬**: 초선·소교·대교는 적 1명을 **1턴 매혹**(행동 불가)시킴.
- **사마의 OP화**: 「화계」 전체 14 피해(총사령관급).
- **SSR 등급 신설**: 사마의·제갈량·관우·조운·하후돈·전위·조조를 **SSR**로 승급(능력치 상향).
  **SSR은 무기 2개 장착** 가능.
- **두 게임 골드 연동**: 히어로즈 블러드 승리 시 포인트만큼 영웅전 골드 **추가**, 패배 시 포인트만큼 **차감**.
- **문서화**: `docs/`에 기능 문서 분리 정리(이 폴더).

## UI/UX 개선 (현재)

삼국 영웅전
- 전투 가운데(공격·방어) 카드 영역을 내부 래퍼로 감싸 **가로 스크롤** 처리 — 카드 4장↑일 때 좌측 끝 카드가 잘리지 않음.
- 지도 화면의 상시 파티 나열을 **👥 장수 / 🗡️ 장비 버튼 + 팝업**으로 정리.
  장비 팝업은 보유 무기와 착용 중인 장수/미장착 수량을 보여줌.

히어로즈 블러드
- 진입 시 **덱 구성 화면부터** 표시(덱 15장 + 판 크기 선택 후 시작).
- **판 크기 3×5 · 4×5 · 5×5** 선택 추가(`ROWS` 가변화, `qb_rows` 저장).
- 종료 정산(영웅전 골드) 메시지를 **정돈된 배지**(`#endGold`)로 재구성.

## 전투 시스템 확장 (현재)

삼국 영웅전
- **치명타(크리티컬)**: 기본 공격이 기본 1% 확률로 **공격력 2배**. 적도 1% 확률로 치명타.
- **치명타 무기 3종 추가**: 청강검(+5%)·의천검(+7%)·자웅일대검(+10%) → 무기 **총 12종**.
- **무기 장착 수 등급화**: **C=0(장착 불가) · R=1 · SR=2 · SSR=3**.
- **MP 소모전화**: MP도 HP처럼 모험 내내 유지. **전투 시작 시 10% 회복**, 전역 평정 시 완전 회복,
  **회복 스킬은 HP 회복량의 1/4만큼 MP도 회복**. 이에 맞춰 적 수치를 하향 재밸런싱(봇 ~98/50/10%).

공통
- 상단 메뉴를 **우측 상단 플로팅 버튼(펼침형)**으로 변경, 패널 정보는 줄바꿈 없이 정렬.
- 히어로즈 블러드 **덱 구성 카드에 강화(영역) 패턴 표시** 추가.

## UI/규칙 다듬기 (현재)

삼국 영웅전
- 지도의 **출진/저잣거리 버튼 가운데 정렬**, 그 아래 **주공 HP·MP·골드 상태** 표시.
- 회복 스킬 MP가 줄던 버그 수정 → **순증가**(소모 없음, round(HP/4)).
- 메뉴에 **📖 게임 가이드** 모달 추가(가독성 있는 플레이 안내).
- 회복 스킬 버프로 약간 쉬워져 봇 클리어율 ~100/60/18%.

히어로즈 블러드
- 첫 화면(랜딩) **설명 업데이트**(판 크기·44종·골드 정산).
- 상대/당신 옆 **덱·패 글자 제거**(아이콘 🃏/✋로 대체).
- **배치 규칙 변경**: 칸의 폰 레벨 **이상**의 등급 카드만 배치 가능(`def.rank >= cell.rank`).
  예) 레벨 2 칸 → 등급 2·3·4·5 카드. 규칙 모달/랜딩/문서 갱신.

## 모드·장비·밸런스 확장 (현재)

- **컬렉션 획득 경로 표시**: 장수/장비 컬렉션에서 항목을 선택하면 보유 여부 + **획득 경로**가 표시됨.
- **모드(노멀→하드→극악)**: 천하통일로 차례 해금(`hw_mode_unlocked`). 하드=적 HP 2배·적장 공격 +50%,
  극악=적 HP 3배·적장 공격 +100%·적장 치명타 10%. 상위 모드 골드 보상↑. 시작 모달에서 모드 선택.
- **자웅일대검**: 장수 컬렉션 100% 완료 보상으로만 지급(`exclusive:'collection'`, 일반 풀 제외).
- **회복 스킬 MP 회복 삭제**: 회복 스킬은 HP만 회복(MP 소모·회복 없음, 무료).

- **명마 장비 4종**(적토마10%·적로/절영/조황비전7%) + **회피(evade)** 능력치 추가 — 적 공격을 확률로 무효화(합산 최대 60%).
- **치명타 장비 +5%**: 치명타가 붙은 모든 장비의 확률을 5%p 상향(자웅일대검 15% 등). 무기 총 **29종**.
- (정리) 기존 유물 적토마 → **오추마**로 개명(명마 장비 적토마와 구분).

## 대장전 보상 개편 · 초기화 · 컬렉션 상세 (현재)

- **대장전 보상 = 그 적장 본인**: 전용 변종("맹장 화웅" 등)을 폐기하고, 보스 격파 시 **그 적장 본인을 일반 장수로 영입**하도록 변경.
  - 기존 로스터에 없던 **화웅·원소·사마염**을 일반 장수(SSR)로 추가, 나머지 5명(하후돈·조조·하후연·육손·사마의)은 **기존 로스터의 그 장수**를 보상으로 지정.
  - 전용 변종 8종(`exclusive:'raid'`) 제거 → 장수 총 **53종**, 전용 장수는 제갈량(QB 3연승)만 남음.
- **데이터 초기화 메뉴**: 영웅전 메뉴에 **🗑️ 데이터 초기화** 추가 — 확인 후 영웅전·대장전 진행 + 컬렉션 + 모드 해금 + 연동 골드 전부 삭제 후 새로고침.
- **장수 컬렉션 상세에 능력치 표시**: 장수 선택 시 병과·HP·공격력·무기 슬롯·스킬을 획득 경로와 함께 표시.

## 대장전 전용화 · 도감 탭 UI · 난이도 재조정 (현재)

- **대장전 보상 장수 8종 전용화**: 화웅·원소·사마염·사마의·조조·하후돈·하후연·육손을 `exclusive:'raid'`로 지정 — 영웅전 일반 보상/상점 풀에서 제외하고, 해당 레이드 보스를 격파해야만 획득. (화웅·원소·사마염은 신규 추가, 나머지 5명은 기존 로스터에서 전용으로 이동.)
- **컬렉션 → 도감 개명·이동**: "장수/장비 컬렉션"을 **도감**으로 개명하고 우측 상단 메뉴에서 제거, 맵 화면 하단 **📖 도감** 버튼으로 이동. 단일 팝업(`codexModal`) 상단의 **📕 장수 도감 / 🗡️ 장비 도감 탭** 전환 방식으로 통합.
- **난이도(`HW_DIFF`) 재조정**: 일반 풀에서 강한 장수가 빠진 만큼 하향 — easy `eHp 0.95 / eAtk 1.04`, normal `eHp 1.02 / eAtk 1.10`, hard `eHp 1.05 / eAtk 1.14`. 밸런스 봇 클리어율 **easy ~98% · normal ~84% · hard ~44%**.

## 시작 장수·연대기·대장전 보상·보스 스킬 (현재)

- **시작 장수 변경**: `HW_STARTERS`가 **전위 · 하후연 · 순욱 · 조인**으로 변경(이전 전위·조인·순유·장비). **하후연**이 시작 장수가 되며 `exclusive:'raid'`가 해제 → **대장전 전용 7종**(화웅·원소·사마염·사마의·조조·하후돈·육손)으로 감소(장수 총 53종 유지).
- **맵 제목 = 연대기 + 모드**: 영웅전 지도 화면 제목을 "🗺️ 진행 경로" → **"📜 연대기"** 로 변경하고 현재 모드를 덧붙임(예: `📜 연대기 · 🟢 노멀`, 🟢 노멀 / 🟠 하드 / 🔴 극악).
- **대장전 격파 골드**: 레이드 보스 **첫 격파** 시 영웅전 골드(`hw_bonus_gold`)를 난이도별 **50 / 100 / 150**(`HW_BOSS[diff].raidGold`) 적립(재격파는 없음). 하후연 보스 보상은 시작 장수와 중복(결과창 "이미 보유").
- **대장전 선택 화면 버튼**: 보스 선택 화면에 **👥 장수 / 🗡️ 장비 / 📖 도감** 버튼 추가(공유 덱 조회 + 도감).
- **보스 스킬·MP(`HW_BOSS`)**: 영웅전·대장전 보스 모두 MP를 보유하고 대응 장수(`HW_COMMANDERS.hero`)의 원래 스킬을 시전. 난이도별 시전 확률 **20 / 35 / 50%**, MP **20 / 35 / 50**, 비용 `2+skill.cost`·MP 재생 없음. 스킬은 주공을 노리며(strike/aoe/multi/heal/shield), **버프는 기본 공격력의 +60%까지만 누적**(초과분은 피해).
- **밸런스**: 보스 스킬 도입으로 봇 클리어율 약 **easy ~96% · normal ~74% · hard ~40~50%**(`HW_DIFF` 배수표는 그대로 유지).

## 소모품·중간보스·11출진·난이도 개편 (현재)

삼국 영웅전 대규모 게임플레이 개편 + 전 게임 난이도 선택 제거.

- **난이도 선택 제거 → 상(hard) 고정**: 첫 페이지(`index.html`)의 상/중/하 선택과 각 게임 메뉴의 난이도 pill 제거. `TCG.getDifficulty()` 기본값 `hard`(내부 테스트용 `?diff=` 오버라이드만 유지). 정상 플레이는 전부 상. (모드 노멀/하드/극악과는 무관.)
- **출진 11회(`SUB_COUNT` 12→11)**: 각 메인 전역이 11 출진 — **11번째 = 적장(보스)**, **5·10번째 = 중간보스**. 맵(연대기) 진행 점에서 **5·10(⚜ 보라)** 과 **11 보스(👑)** 를 구분 강조.
- **중간보스(`HW_MID`·`HW_MID_SKILLS`)**: 정예 기반 **HP ×2.3 / 공격 ×1.3**, **MP 보유**(`min(50, 20+main*4)`), `skillChance` 0.45로 **아군 카드(장수) 상태이상 스킬** 시전. 4종(미혹의 진=매혹·환혼술=혼란·독무 살포=중독 val4·사신 강림=봉인)을 스테이지별 순환. 격파 시 **장수 영입 보상(골드 1.5배)**. 유닛에 "중간보스"/"적장" 태그.
- **상태이상(내 카드 대상)**: 혼란·매혹=그 카드 1턴 행동 불능(방어 시 블록 0), 중독=턴마다 주공 지속 피해(턴당 1 감소·해독초로 해제), 사신(봉인)=카드 1장 이번 전투 동안 덱 제외(전투당 1회). `combat.cstat`(uid→{stun,poison,cause})·`combat.sealed`로 추적. (적에게 거는 매혹/혼란은 기존대로 적 행동 불가.)
- **소모성 아이템(`HW_CONSUMABLES` 6종)**: 회복약(🧪 HP+45)·마력약(💧 MP+20)·해독초(🌿 중독 해제)·안신향(🔔 혼란·매혹 해제)·전투주(🍷 2턴 전군 공격+5)·철벽부(🛡️ 방어막+8). 전투 중 **턴당 1개**·**최대 5칸**(`HW_ITEM_MAX`). 저잣거리에서 **랜덤 2개**(각 22골드). 하단 아이템 바(`#itemBar`) 사용. 저장 `hw_save.items`(`run.items`).
- **혼란 스킬 장수 3명**: 가후(「교란계」)·주유(「반간계」)·순유(「허허실실」) — 모두 `confuse`, 적 1명 1턴 혼란, MP 3. 순유는 치유 풀에서 빠짐.
- **연출 강화**: 치명타=화면 플래시 + 이중 링 + 강한 흔들림(`shake-huge`) + 「💥 치명타!」 배너. 스킬 발동=「✨ 장수 「스킬」」 배너 + 플래시 + 링.
- **난이도 수치 상향(`HW_DIFF`)**: easy `1.00/1.09` · normal `1.07/1.15` · hard `1.10/1.18`. 봇 클리어율 약 **하 ~74% / 중 ~54% / 상 ~22%**(중간보스·상태이상 반영). 게임은 상 고정이고 봇은 소모품을 쓰지 않아 실제 플레이어 체감은 더 높음.

## 다회 공격 연출 분리 (현재)

- **2회 연속·무작위 N회 공격 연출 분리**: 다회 공격(칠성검 등 2회 연속, 황충·여몽 등 무작위 N회 스킬)을 동기 루프에서 **타격별 타이머**(약 0.2~0.24초)로 변경 — 타격마다 피해·연출·HP 갱신이 따로 보입니다. 영웅전(`heroes.js`)·대장전(`daejang.js`) 모두 적용. 연출 동안 입력 잠금(`combat.busy`)으로 중복 행동 방지.
- (수정) 대장전에서 처리되지 않던 **혼란(confuse) 스킬** 분기 추가 — 가후·주유·순유 스킬이 레이드에서도 동작(보스 1턴 행동 불가).
- **대장전 보스 행동 불가 면역**: 레이드 보스는 **혼란·매혹(행동 불가)에 면역**. 해당 스킬(가후·주유·순유·초선·소교·대교)은 대장전에서 스킬 버튼이 비활성화(🛡 면역)되어 사용 불가(MP 낭비 없음). 독·피해 스킬은 그대로 적용.

## 영웅전 카드 방어 제거 · 전투 내역 확대 (현재)

- **"턴 종료 시 남기면 방어" 기능 제거**: 영웅전에서 가운데에 남은 카드는 이제 **방어(블록) 없이 그냥 버려집니다**(사용한 더미로 이동). 카드의 🛡 방어 수치 표시와 "🎴 가운데 카드: …" 안내 문구도 제거. 주공 블록은 **방어막 스킬·철벽부 아이템·유물**로만 올립니다. (대장전은 기존 방어 유지.)
- **전투 내역(combat log) 확대**: 가로를 **꽉 채우고**(full width) 높이를 약 **3배**(min 120 / max 168px)로 키움. 상단 안내 문구(energy) 영역 제거.
- **재밸런싱**: 카드 방어가 사라진 만큼 적 공격(`HW_DIFF.eAtk`)을 하향(easy 0.84 / normal 0.93 / hard 1.01), 진행 공격 보정 `prog·0.007→0.005`, 승리 회복 `6%→약 11.5%`. 봇 클리어율 약 **하 ~95% / 중 ~63% / 상 ~32%**.

## 다회 공격 스킬 조정 · 상점 상시 · 버전/변경내역 (현재)

- **다회 공격 스킬 데미지 분할**: `multi` 스킬은 타격당 **기본 공격력의 1/N(반올림)** — 2회=½·3회=⅓·4회=¼…(총합 ≈ 기본 공격력). 영웅전·대장전 모두 적용. **단, 장비의 2회 연속(doubleStrike)은 타격마다 풀 피해** 그대로. (multi가 N×풀피해이던 과대 위력을 정상화 → 봇 상 ~22%.)
- **저잣거리 상시 입장**: 영웅전 지도에서 저잣거리를 **언제나** 들어갈 수 있게 변경(`run.stageShopped` 잠금 제거, 방문 시마다 품목 재구성).
- **첫 화면 버전·변경 내역**: 홈 하단에 **버전 배지(v1.0)** 와 **📜 변경 내역** 버튼 추가 — 클릭 시 팝업으로 변경 사항을 표시.
- **캐시 버스터 갱신**: 정적 자원 쿼리 `?v=20260616a → 20260617a`(그동안의 변경분이 캐시 없이 배포되도록).

## 히어로즈 블러드 배치 규칙 반전 · 카드 정보 팝업 (현재)

- **배치 규칙 반전**: 카드는 칸의 폰 레벨 **이하**의 등급만 놓을 수 있게 변경(`canPlace`: `def.rank >= cell.rank` → `def.rank <= cell.rank`). 예) 레벨 4 칸 → 등급 1·2·3·4 / 레벨 2 칸 → 등급 1·2. 초반엔 레벨 1 칸뿐이라 등급 1 카드부터 시작해 강화로 레벨을 올려 높은 등급을 전개. (규칙 모달·문서 갱신.)
- **보드 카드 정보 팝업**: 보드에 놓인 카드를 탭하면 등급·무력(기본/효과)·능력·강화 패턴을 팝업(`#cardModal`)으로 표시 — **상대 카드도** 확인 가능.

## 행동 불가 타이밍 변경 · 적 스킬 빈도 ↓ (현재)

- **행동 불가(혼란·매혹) 타이밍 변경**: 상태이상에 걸린 카드는 **턴과 무관하게, 다음 출진(가운데 등장) 시 1회 행동 불가**가 됩니다. 덱(뽑을 더미)에 묻혀 있으면 상태가 **유지**되다가, 실제로 출진해 그 턴을 보내야 1회 소모되어 회복됩니다. (기존: 턴마다 자동 감소 → 변경: 출진한 카드만 소모.)
- **적 스킬 시전 빈도 약 30% 감소**: `HW_BOSS.skillChance` 하 0.20→**0.14** · 중 0.35→**0.25** · 상 0.50→**0.35**, `HW_MID.skillChance` 0.45→**0.32**. (영웅전·대장전 보스 공통.)

## 행동 불가 비중첩 · 스킬 치명타 · 여포 · 대장전 간격 (현재)

- **행동 불가(매혹·혼란) 비중첩**: 다시 걸어도 누적되지 않고 **마지막 1개만 적용**(매혹↔혼란 상호 배타). 항상 1회만 행동 불가.
- **스킬 공격에도 치명타 적용**: 기본 공격뿐 아니라 strike·aoe·다회 스킬에도 치명타(피해 2배). strike/aoe는 1회, 다회는 타격마다 판정. 영웅전·대장전 공통.
- **여포(SSR, `exclusive:'special'`) 추가** — 장수 총 54종. 획득: **첫 적장 화웅을 주공 풀 HP로 격파** 또는 **노멀 모드 천하통일**(둘 중 하나). 일반 보상/상점 제외. 능력치 atk 15·「천하무쌍」 strike 18.
- **대장전 선택 화면 간격**: 상단 장수/장비/도감 버튼과 보스 목록 사이 간격(`#selectScreen .map-collect` margin-bottom)을 넓힘.
