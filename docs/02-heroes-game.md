# 02 · 삼국 영웅전 (로그라이크 카드 전투)

파일: `heroes.html` · `js/heroes.js` · `js/heroes_data.js` · `css/heroes.css`

## 한 줄 요약

플레이어는 **주공(나)** 입니다. 장수 카드로 적을 물리치며 **주공의 HP**를 지키고,
**8개 역사 전역 × 각 11회 출진**을 모두 평정해 **천하통일 → 엔딩 크레딧**을 보는 것이 목표입니다.

---

## 1. 주공(나) — HP / MP

- 플레이어 자신이 **주공**이며 **시작 HP 200 · MP 50**을 가집니다. (`HW_LORD = { hp:200, mp:50 }`)
- **적군은 개별 장수가 아니라 주공의 HP를 공격**합니다. 주공의 HP가 0이 되면 패배(💀 주공 전사).
- **HP는 모험 내내 유지**됩니다(로그라이크 소모전):
  - 전투 승리 시 최대 HP의 **약 11.5% + 유물 보정**만큼 회복(카드 방어 제거 보정으로 상향).
  - **메인 전역을 평정(적장 격파)** 하면 **완전 회복**.
  - 저장 시 `run.lordHp`로 보존.
- **MP도 HP처럼 모험 내내 유지**됩니다(스킬 비축이 중요한 소모전 자원). 스킬 사용 시 **3~5 소모**.
  - 비용 공식: `skillMp = 2 + skill.cost` (skill.cost 1~3 → MP 3~5). MP가 부족하면 스킬 버튼이 비활성화됩니다.
  - **전투 시작 시 최대 MP의 10% 회복**, **전역 평정 시 완전 회복**.
  - **회복 스킬은 HP만 회복**합니다(MP 소모·회복 없음).
  - 저장 시 `run.lordMp`로 보존.
- 주공의 **최대 HP/MP는 무기로 증가**시킬 수 있습니다(아래 무기 항목).

## 1-b. 치명타(크리티컬)

- 기본 공격은 **기본 1% 확률**로 **공격력 2배** 피해(치명타)가 들어갑니다.
- **치명타 확률 +무기**(청강검 +10% · 의천검 +12% · 자웅일대검 +15%)로 확률을 높일 수 있습니다 — 장착한 장수의 공격에 적용.
- **적도 1% 확률로 치명타** 공격을 합니다. 치명타는 금색 연출 + 「치명타!」 표시로 구분됩니다.

## 1-c. 적장(보스) 스킬·MP

- 적장(보스)도 **MP를 보유**하고 **자기 장수 스킬**을 사용합니다 — 영웅전 스테이지 보스와 대장전 레이드 보스 모두 해당합니다.
- 각 `HW_COMMANDERS` 항목은 `hero` 필드로 **대응 장수**와 연결되며, 보스는 그 장수의 원래 `skill`을 사용합니다.
- 보스는 공격할 때 난이도별 확률 `HW_BOSS[diff].skillChance`(하 20% / 중 35% / 상 50%)로 기본 공격 대신 **스킬을 시전**합니다(MP가 충분할 때). 게임은 **상 고정**이므로 실제 플레이에서는 상(50%)입니다.
- 보스 MP 풀은 `HW_BOSS[diff].mp`(하 20 / 중 35 / 상 50). 스킬 비용은 `2 + skill.cost`이며 **MP 재생이 없어** 시전 횟수가 제한됩니다.
- **중간보스(5·10 출진)** 도 MP를 보유하고 **아군 카드(장수) 상태이상 스킬**을 시전합니다(아래 §6·「중간보스」 참고).
- 보스 스킬은 **주공**을 노립니다:
  - `strike` — 주공이 `보스 공격 + ⌊val/2⌋` 피해(치명타 가능).
  - `aoe` — 주공이 `⌊val·0.6⌋ + ⌊atk·0.3⌋` 피해.
  - `multi` — `⌊atk·0.5⌋` 피해를 `val`회 타격.
  - `buff` — 보스 공격력 +`val`, **기본 공격력의 +60%까지만 누적**(상한 초과분은 피해로 전환).
  - `heal` — 보스 자가 회복.
  - `shield` — 보스 블록.
- 적장→장수 스킬 대응: 화웅→관문 수장(strike) · 원소→사세삼공(buff) · 하후돈→발시담정(strike) · 조조→간웅(buff) · 하후연→질풍사격(multi) · 육손→이릉대화(aoe) · 사마의→화계(aoe) · 사마염→삼국 통일(aoe).
- 데이터: `HW_BOSS = { easy:{mp:20,skillChance:0.20,raidGold:50}, normal:{mp:35,skillChance:0.35,raidGold:100}, hard:{mp:50,skillChance:0.50,raidGold:150} }`.

## 2. 카드 전투 — 3구역 보드

하단이 세 구역으로 나뉩니다.

```
[ 왼쪽: 뽑을 카드 더미 ]   [ 가운데: 공격·방어 덱 (3장) ]   [ 오른쪽: 사용한 카드 더미 ]
```

- **덱 = 보유 장수 전체**(1장수 = 1카드). 전투 시작 시 전원을 섞어 뽑을 더미를 구성.
- **가운데 카드를 쓰면 공격**(기본 공격 또는 스킬). **턴 종료 시 남은 카드는 그냥 버려집니다**(사용한 더미로 이동) — 예전의 "남기면 방어" 기능은 제거됐습니다.
- 라운드마다 가운데를 **3장**까지 자동 보충(유물 `대장군 깃발`로 +1). 카드가 칸 폭을 넘으면 가운데 영역이 **가로 스크롤**되어 좌측 끝 카드도 잘리지 않고 모두 접근할 수 있습니다.
- **뽑을 더미가 비면** 사용한 더미를 **랜덤 셔플**해 보충합니다.
- 장수는 전투 중 **죽지 않습니다**(HP는 주공만 보유). 카드가 더미를 순환할 뿐입니다.

### 행동 — 기본 공격 / 스킬

| 행동 | 효과 |
|------|------|
| **기본 공격** | 대상 적에게 `effAtk`(공격력+무기+버프) 피해. 무기로 2회 연속/독 부여 가능. |
| **스킬** | 카드별 고유 스킬. **MP 소모**. 단일·전체·무작위·회복·방어막·고무·매혹·혼란 등. |
| **턴 종료** | 가운데에 남은 카드는 **버려집니다**(방어 기능 없음). 주공 블록은 **방어막 스킬·철벽부 아이템·유물**로만 올립니다. |

- 주공 **블록**은 다음 적 턴의 피해를 먼저 흡수하고 라운드 시작 시 0으로 초기화됩니다(1라운드는 유물 `startBlock` 유지). 카드 방어가 사라진 만큼 방어막 스킬·아이템 비중이 커졌습니다.
- 적은 다음 행동을 머리 위에 **예고(intent)** 합니다(⚔️ 단일 / 💥 광역).

### 스킬 타입

| type | 동작 | 대상 |
|------|------|------|
| `strike` | 단일 적에게 `effAtk + val` 피해 | 적 1명(타겟 선택) |
| `aoe` | 모든 적에게 `val` 피해 | 전체(자동) |
| `multi` | 무작위 적을 `val`회, 각 `effAtk` 피해 | 무작위(자동) |
| `heal` | **주공** `val` 회복 (제갈량은 방어막 +5 추가) | 주공(자동) |
| `shield` | **주공** 블록 +`val` | 주공(자동) |
| `buff` | 전군 공격력 +`val` (해당 전투 동안) | 전군(자동) |
| `charm` | 적 1명을 **`val`턴 매혹**(행동 불가) | 적 1명(타겟 선택) |
| `confuse` | 적 1명을 **1턴 혼란**(다음 자기 턴 행동 불가) | 적 1명(타겟 선택) |

> 회복/방어막/고무 계열은 (옛 파티 모델의 흔적으로 설명문에 "아군"이 남아 있어도)
> 모두 **주공에게 적용**됩니다.

> **혼란(confuse) 스킬 장수 3명**: 가후(「교란계」)·주유(「반간계」)·순유(「허허실실」) — 모두 적 1명을 1턴 혼란(행동 불가)시키며 MP 3입니다. (순유는 더 이상 치유 장수가 아닙니다.)

## 3. 매혹(Charm)

- **초선·소교·대교**의 스킬은 **매혹**입니다.
- 적 1명을 지정해 **1턴 매혹** → 매혹된 적은 다음 자기 턴에 **행동하지 못하고 쉽니다**(💤).
- 적 머리 위 예고가 `💤 매혹`으로 바뀌고, 카드/유닛에 💗 표시가 붙습니다.
- 강력한 적장(보스)을 한 턴 묶는 용도로 유용합니다.

## 3-b. 상태이상 (내 카드 = 장수 대상)

**중간보스**가 거는 상태이상은 **적이 아니라 내 카드(장수)** 에 적용됩니다. 전투는 이를 `combat.cstat`(uid→{stun,poison,cause})와 `combat.sealed`로 추적합니다.

| 상태이상 | 효과 |
|----------|------|
| **혼란/매혹** | 해당 카드가 **1턴 행동 불능**(공격·스킬 불가, 방어 시 블록 0). |
| **중독** | **턴마다 주공이 지속 피해**(턴당 수치 1씩 감소, 해독초로 즉시 해제). |
| **사신(봉인)** | 카드 1장을 **이번 전투 동안 덱에서 제외**(전투당 1회). |

> 내가 적에게 거는 매혹/혼란은 기존대로 **적의 다음 자기 턴 행동 불가**입니다(§3 참고). 위 표는 중간보스가 **내 장수에게** 거는 경우입니다.

## 3-c. 소모성 아이템 (`HW_CONSUMABLES`, 6종)

- 전투 중 **턴당 1개**만 사용할 수 있으며, **최대 5칸**까지 보유합니다(`HW_ITEM_MAX`).
- 전투 화면 하단 **아이템 바(`#itemBar`)** 에서 사용합니다. 보유 목록은 `run.items`(저장 `hw_save.items`).
- **저잣거리(상점)** 에서 매 방문 시 **랜덤 2개**(각 22골드)를 판매합니다.

| 아이템 | 효과 |
|--------|------|
| 회복약 🧪 | 주공 HP +45 |
| 마력약 💧 | MP +20 |
| 해독초 🌿 | 중독 해제 |
| 안신향 🔔 | 혼란·매혹 해제 |
| 전투주 🍷 | 2턴간 전군 공격력 +5 |
| 철벽부 🛡️ | 주공 방어막 +8 |

## 4. 무기 (장수당 장착)

- 장수마다 무기를 장착할 수 있습니다. 무기는 **삼국지 아이템**이며, 효과로 전투력을 강화합니다.
- **등급별 장착 수**: **C=0개(장착 불가) · R=1 · SR=2 · SSR=3**(`weaponSlots`).
- 장착/해제는 **장수 상세(미니 초상화 탭)** 에서 합니다. 같은 무기를 동시에 두 장수에 장착할 수는 없습니다.

| 무기 | 효과(`effect`) |
|------|------|
| 장팔사모 🔱 | 공격력 +3 (`atk`) |
| 청룡언월도 🗡️ | 공격력 +4 |
| 방천화극 ⚔️ | 공격력 +5 |
| 칠성검 🌟 | 기본 공격 **2회 연속** (`doubleStrike`) |
| 맹덕신서 📕 | 공격 시 적에게 **독 +2** (`poison`) |
| 태평요술서 ☯️ | 공격 시 적에게 **독 +3** |
| 청강검 🔪 | **치명타 확률 +5%** (`crit`) |
| 의천검 🗡️ | **치명타 확률 +7%** |
| 자웅일대검 ⚔️ | **치명타 확률 +10%** |
| 전국옥새 🟨 | **주공 최대 HP +60** (`lordHp`) |
| 손자병법 📜 | **주공 최대 MP +20** (`lordMp`) |
| 한수정후인 🟥 | 주공 최대 HP +30, MP +10 |

> 위는 대표 일부입니다. **무기는 총 29종**(명마=회피 4종 포함) — 전체 표는 [04-data-reference.md](04-data-reference.md) 참고.
> **회피(명마)**: 적토마(10%)·적로/절영/조황비전(7%) — 적 공격을 확률로 무효화(합산 최대 60%).
> **모드**: 노멀→(천하통일)→하드(적HP2배·적장공격+50%)→극악(적HP3배·적장공격+100%·적장치명타10%) 순으로 해금.

- **독(poison)**: 적의 턴 시작 시 누적 독 수치만큼 피해. 여러 번 부여하면 누적됩니다.
- **2회 연속**: 기본 공격이 두 번 적중(적이 먼저 죽으면 중단).
- 무기 효과는 합산됩니다(예: SSR이 청룡언월도+방천화극+장팔사모를 끼면 공격력 +12).

## 5. 파티 / 장수 수집 · 출진 구성

- 시작 장수 **4명**: 전위 · 하후연 · 순욱 · 조인 (`HW_STARTERS`).
- 각 메인 전역은 **11회 출진**(`SUB_COUNT`)으로 구성됩니다 — **11번째 출진 = 적장(보스)**, **5·10번째 출진 = 중간보스**.
- 보상/저잣거리로 장수를 영입. **파티 최대 30명**(`MAX_PARTY`, =덱 용량). 가득 차면 **교체**.
- **중복 수집 불가**: 이미 보유한 장수는 보상/상점 후보로 제시되지 않습니다(`ownedHeroIds`).
  모든 장수를 보유한 경우 장수 보상은 **보물(무기)로 대체**됩니다.
- **장수 총 53종** — 자세한 표는 [04-data-reference.md](04-data-reference.md) 참고. **제갈량**(QB 3연승)과 **대장전 전용 7종(화웅·원소·사마염·사마의·조조·하후돈·육손)** 만 획득 경로가 제한되고, 나머지는 전투 보상/상점에서 일반 영입할 수 있습니다. 대장전 전용 7종은 `exclusive:'raid'`로 일반 풀에서 빠져 해당 레이드 보스 격파로만 얻습니다. (**하후연**은 전용에서 풀려 **시작 장수**가 됐습니다.)

### 데이터 초기화

- 우측 상단 메뉴의 **🗑️ 데이터 초기화** — 확인 후 삼국 영웅전·삼국 대장전의 진행 저장, 장수/장비 컬렉션, 모드 해금, 연동 골드를 모두 삭제하고 페이지를 새로고침합니다(`hw_save` · `hw_raid_cleared` · `hw_mode_unlocked` · `hw_collected_heroes` · `hw_collected_weapons` · `hw_grant_heroes` · `hw_bonus_gold` 제거).

### 도감 상세

- 도감은 지도(맵) 화면 하단의 **📖 도감** 버튼으로 열리는 **단일 팝업**(`codexModal`)이며, 상단의 **📕 장수 도감 / 🗡️ 장비 도감 탭**으로 전환합니다. (예전처럼 우측 상단 메뉴에 별도 항목으로 있던 "장수 컬렉션/장비 컬렉션"은 제거됐습니다.)
- **📕 장수 도감**에서 장수를 탭하면 병과·HP·공격력·무기 슬롯 수·스킬과 **획득 경로**가 함께 표시됩니다. 대장전 전용 장수는 경로에 `👹 삼국 대장전 — <적장> 격파 보상 (대장전에서만 획득)`이 표시됩니다.
- **🗡️ 장비 도감**도 동일하게 효과·획득 경로를 표시합니다.

### 지도 화면의 장수 / 장비 보기

지도(맵) 화면 하단에는 세 버튼이 있습니다(파티 목록을 항상 나열하지 않고 팝업으로 정리).

- **👥 장수**: 수집한 장수 전체를 팝업 그리드로 표시. 장수를 탭하면 상세/무기 장착 모달로 이동.
- **🗡️ 장비**: 보유한 무기 목록과 **각 무기를 누가 착용 중인지/미장착 수량**을 표시.
- **📖 도감**: 위 도감 팝업(장수/장비 탭)을 엽니다.

### 등급(rarity)

`C < R < SR < SSR`. **SSR**은 최고 등급으로 **무기 3개 장착**이 가능하며 능력치가 높습니다.

- SSR 10종: **전위 · 사마의 · 관우 · 제갈량 · 조운 · 조조 · 하후돈 · 화웅 · 원소 · 사마염**
- **사마의**는 총사령관급으로 의도적으로 강하게(OP) 조정 — 「화계」로 **전체 14 피해**(MP 5).

## 6. 전역(캠페인) 구조

- **8개 역사 전역**(`HW_STAGES`)을 차례로 평정합니다. 각 전역은 **11회 출진**(`SUB_COUNT`).
  - 마지막 11번째 출진은 그 시대의 **적장(보스)전**.
  - **5·10번째 출진은 중간보스(중간 보스)전**(아래 참고).
  - 적장을 격파하면 다음 메인 전역이 **해금**되고 주공이 완전 회복됩니다.
- **뒤로 갈수록 난이도 상승**: 적 HP/공격이 전체 진행도 `prog = main*11 + sub`에 비례해 증가
  (`hpM = eHp·(1+prog·0.020)`, `atkM = eAtk·(1+prog·0.005)`).
- 지도(맵) 화면 제목은 **"📜 연대기"** 이며 뒤에 현재 모드가 붙습니다 — 예: `📜 연대기 · 🟢 노멀`(🟢 노멀 / 🟠 하드 / 🔴 극악).
- **진행 표시 강조**: 맵(연대기)의 출진 진행 점에서 **5·10 출진(⚜ 보라)** 과 **11출진 보스(👑)** 를 구분해 강조합니다.

### 중간보스 (중간 보스, 5·10 출진)

- 정예 적을 기반으로 **HP ×2.3 / 공격 ×1.3** 강화한 적(`HW_MID`)입니다. 유닛에 **"중간보스"** 태그가 붙고, 11번째는 **"적장"** 태그가 붙습니다.
- **MP를 보유**(스테이지별 20~50, `= min(50, 20 + main*4)`)하고, `HW_MID.skillChance`=0.45 확률로 **아군 카드(장수) 상태이상 스킬** 1개를 시전합니다.
- 4종 스킬(`HW_MID_SKILLS`)을 스테이지별로 하나씩 순환 사용합니다 — **미혹의 진(매혹)·환혼술(혼란)·독무 살포(중독, val 4)·사신 강림(봉인)**.
- 상태이상은 **내 장수에게** 적용됩니다(§3-b 참고).
- **격파 시 장수 영입 보상**(골드 1.5배).

8개 전역(연도/적장):
반동탁 연합군(191·화웅) → 관도대전(200·원소) → 장판파(208·하후돈) → 적벽대전(208·조조) →
정군산(219·하후연) → 이릉대전(222·육손) → 제갈량의 북벌(228~234·사마의) → 위·촉·오 멸망(263~280·사마염).

### 적 구성

- 일반 출진: 1~3명(서브가 진행될수록 증가, 후반엔 정예 1명 포함).
- 보스 출진: 일반 2명 + **적장 1명**.
- 적 데이터: `HW_ENEMIES`(basic/elite/boss), 적장: `HW_COMMANDERS`.

## 10. 난이도

> **난이도 선택은 제거됐습니다.** 첫 페이지(`index.html`)의 상/중/하 선택이 사라졌고, 각 게임 우측 상단 메뉴의 난이도 pill도 표시되지 않습니다. `TCG.getDifficulty()`는 기본값 **상(hard)** 을 반환하며(내부 테스트용 URL `?diff=` 오버라이드만 남음), **정상 플레이는 전부 상(hard) 고정**입니다. (모드 노멀/하드/극악 진행과는 무관합니다.)

`HW_DIFF`로 적 HP/공격, 골드, 시작 골드를 조정합니다(내부 수치).

| 난이도 | 적 HP | 적 공격 | 골드 | 시작 골드 |
|--------|-------|---------|------|-----------|
| 하(easy)   | ×1.00 | ×0.84 | ×1.4 | 70 |
| 중(normal) | ×1.07 | ×0.93 | ×1.0 | 50 |
| 상(hard)   | ×1.10 | ×1.01 | ×0.9 | 40 |

> 정확한 수치는 `HW_DIFF`(`heroes_data.js`)이며 전투 모델 변경마다 재튜닝됩니다.
> 카드 방어(턴 종료 시 남긴 카드 블록) 기능이 제거되어, 적 공격(eAtk)을 하향하고 승리 회복을 ~11.5%로 올려 재밸런싱했습니다. 자동 플레이(봇) 기준 대략적 클리어율: **하 ~95% / 중 ~63% / 상 ~32%**. 게임은 **상 고정**이고 봇은 소모품을 쓰지 않으므로, 소모품·방어막을 활용하는 실제 플레이어 체감은 더 높습니다.
> 측정·튜닝 방법은 [06-testing-balance.md](06-testing-balance.md) 참고.

## 11. 연출(FX)

베기·타격·파티클·링·반짝임·투사체·배너·화면 흔들림·사망 연출을 `#fxLayer`에 렌더합니다.
이 레이어는 `renderCombat()`의 innerHTML 재렌더에도 살아남도록 분리되어 있습니다.

- **치명타 강화**: 화면 플래시 + 이중 링 + 강한 흔들림(`shake-huge`) + 「💥 치명타!」 배너로 크게 표현합니다.
- **스킬 발동 강화**: 「✨ 장수 「스킬」」 배너 + 플래시 + 링으로 더 잘 보이게 표시합니다.
