# 08 · 삼국 대장전 (레이드 보스전)

파일: `daejang.html` · `js/daejang.js` (데이터·전투 규칙은 `heroes_data.js`/영웅전과 공유, `css/heroes.css` 재사용)

## 개요

삼국 영웅전의 **덱(파티)을 그대로 공유**해, 8명의 적장을 **거대 HP의 레이드 보스**로 토벌하는 모드입니다.
보스를 격파하면 그 적장을 **전용 장수**로 획득하며, 이 장수들은 **오직 대장전에서만** 얻을 수 있습니다.

## 덱 공유

- `localStorage['hw_save']`(영웅전 진행 저장)의 파티를 그대로 불러와 덱으로 사용합니다.
- 영웅전 진행이 없으면 **시작 장수 4명**으로 시작합니다.
- 무기·등급·공격력 등 모든 스탯과 규칙이 영웅전과 동일합니다(주공 HP/MP·치명타·매혹·회복 스킬 MP 회복 등).

## 순차 진행 (잠금 해제)

- 보스는 **순서대로** 도전합니다 — 첫 보스(화웅) 외에는 **직전 보스를 격파해야** 잠금이 풀립니다(`raidUnlocked(i)`).
- 잠긴 보스는 이름/보상이 `???`로 가려지고 도전 버튼이 비활성화됩니다.
- 진행 상태는 `hw_raid_cleared`(격파한 보스 키 목록)에 **저장**되어 다음 방문에도 유지됩니다.

## 레이드 보스 (`HW_RAID`)

- 8개 보스 = 영웅전 8개 스테이지의 적장(`HW_COMMANDERS`).
- 보스 HP = `적장 기본 HP × hpMult(12)`, 공격 = `기본 공격 × atkMult(1.25) × 난이도`.
  메인 스테이지 보스보다 **여러 배 큰 HP**로 등장합니다.
- 단일 거대 보스전 — 광역/연속/강타/매혹 스킬 모두 보스에게 적용됩니다(매혹은 보스를 1턴 묶어 강력).

## 전용 장수 획득 (대장전 한정)

| 레이드 보스 | 보상 장수(SSR, exclusive:'raid') |
|------|------|
| 화웅 | 맹장 화웅 |
| 원소 | 하북패자 원소 |
| 하후돈 | 독안룡 하후돈 |
| 조조 | 위왕 조조 |
| 하후연 | 신행 하후연 |
| 육손 | 대도독 육손 |
| 사마의 | 총사령관 사마의 |
| 사마염 | 진 무제 사마염 |

- 격파 시 보상 장수 id가 `hw_grant_heroes`에 추가되고, 컬렉션(`hw_collected_heroes`)에도 기록됩니다.
- 삼국 영웅전 지도 진입 시 `applyPendingGrants()`가 이를 파티에 합류시킵니다(자리가 없으면 대기).
- 이 장수들은 `exclusive:'raid'` 플래그로 **영웅전의 일반 보상/상점 풀에서 제외**됩니다.
- 격파한 보스는 `hw_raid_cleared`에 기록되어 목록에 ✔ 표시됩니다(재도전 가능, 보상은 1회).

## 다른 전용 장수

- **제갈량**(`exclusive:'qb'`)은 히어로즈 블러드 **3연승**으로만 획득 — [03 문서](03-queensblood-game.md) 참고.
