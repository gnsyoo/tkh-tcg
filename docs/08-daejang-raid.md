# 08 · 삼국 대장전 (레이드 보스전)

파일: `daejang.html` · `js/daejang.js` (데이터·전투 규칙은 `heroes_data.js`/영웅전과 공유, `css/heroes.css` 재사용)

## 개요

삼국 영웅전의 **덱(파티)을 그대로 공유**해, 8명의 적장을 **거대 HP의 레이드 보스**로 토벌하는 모드입니다.
보스를 격파하면 그 적장을 **영입**합니다 — 별도의 전용 변종("맹장 화웅" 등)이 아니라 **그 장수 본인(화웅 등)** 을 그대로 얻습니다. 이 8명은 `exclusive:'raid'`로 **오직 대장전에서만** 획득할 수 있습니다(영웅전 일반 보상/상점 제외).

## 덱 공유

- `localStorage['hw_save']`(영웅전 진행 저장)의 **수집 장수 + 출진 덱(`deck`)** 을 그대로 불러옵니다.
- 전투는 **출진 덱(선택한 10~30장)** 으로 진행합니다(`activeDeckUids`). **👥 장수** 팝업에서 카드를 탭해 덱을 편성할 수 있고, 변경은 `hw_save.deck`에 저장되어 **영웅전과 공유**됩니다(무기 장착은 영웅전에서).
- 영웅전 진행이 없으면 **시작 장수 4명**으로 시작합니다(덱도 4명 자동 편성).
- 무기·등급·공격력·**스킬 규칙이 영웅전과 동일**합니다(회복/방어막=주공, 모든 스킬 MP 소모, 「아군 1명 공격력+」=출진 카드 1장 1턴/전군 버프 1회, 매 턴 MP 재생).
- **소모성 아이템 사용**: 전투 하단 아이템 바에서 영웅전과 **공유하는 소모품(`hw_save.items`)** 을 턴당 1개 사용(HP/MP 회복·전투주·방어막). 상태이상 해제 아이템은 대장전에 해당 상태가 없어 무효.

## 순차 진행 (잠금 해제)

- 보스는 **순서대로** 도전합니다 — 첫 보스(화웅) 외에는 **직전 보스를 격파해야** 잠금이 풀립니다(`raidUnlocked(i)`).
- 잠긴 보스는 이름/보상이 `???`로 가려지고 도전 버튼이 비활성화됩니다.
- 진행 상태는 `hw_raid_cleared`(격파한 보스 키 목록)에 **저장**되어 다음 방문에도 유지됩니다.

## 레이드 보스 (`HW_RAID`)

- 8개 보스 = 영웅전 8개 스테이지의 적장(`HW_COMMANDERS`).
- 보스 HP = `적장 기본 HP × hpMult(12)`, 공격 = `기본 공격 × atkMult(1.65) × 난이도`(기본 1.25 대비 **약 +32%** = 10% + 20% 누적 상향).
  메인 스테이지 보스보다 **여러 배 큰 HP**로 등장합니다.
- 단일 거대 보스전 — 광역/연속/강타 스킬은 보스에게 적용됩니다. **혼란·매혹(행동 불가)에는 면역**(아래 스킬 치환 참고).
- **보스 스킬 2종·MP**: 레이드 보스는 **MP(난이도별 20/35/50 에서 −20 → 0/15/30)** 와 **스킬 2종**(`HW_COMMANDERS[key].skills`)을 갖습니다 — **① 공격 또는 방어** + **② 회복 또는 행동 불가**. 공격 시 난이도별 확률(20/35/50%)로 둘 중 하나를 시전합니다(HP가 낮으면 회복/방어를 우선, `pickBossSkill`). MP 재생 없음.
  - **행동 불가(confuse/charm) 보스 스킬**: 주공을 `val`턴 행동 불가로 만듭니다 — 그 턴 동안 카드로 **공격/스킬을 쓸 수 없고**(턴 종료=방어만 가능), `c.lordStun` 턴으로 관리됩니다.
- **혼란·매혹 스킬 치환(대장전 한정)**: 보스가 면역이라, 내 카드의 **혼란→「계책」(전군 공격+5 버프)**, **매혹→「구휼」(주공 회복 12)** 로 전투 중에만 교체됩니다(`raidSkill`). 도감에는 원래 스킬이 그대로 표시됩니다.
- **보스 인물화**: 레이드 보스 초상화는 대응 장수와 **같은 절차적 얼굴**(hero id 시드)로 표시됩니다(`bossFace`).

## 보상 장수 획득 (그 적장 본인을 영입)

| 레이드 보스 | 영입 장수(ID) |
|------|------|
| 화웅 | 화웅 (`huaxiong`) |
| 원소 | 원소 (`yuanshao`) |
| 하후돈 | 하후돈 (`xiahoudun`) |
| 조조 | 조조 (`caocao`) |
| 하후연 | 하후연 (`xiahouyuan`) |
| 육손 | 육손 (`luxun`) |
| 사마의 | 사마의 (`warlock`) |
| 사마염 | 사마염 (`simayan`) |

- 화웅·원소·사마염은 기존 로스터에 없던 적장이라 **일반 장수로 신규 추가**됐고, 나머지 5명은 기존 로스터의 그 장수입니다.
- 격파 시 영입 장수 id가 `hw_grant_heroes`에 추가되고, 컬렉션(`hw_collected_heroes`)에도 기록됩니다.
- 삼국 영웅전 지도 진입 시 `applyPendingGrants()`가 이를 파티에 합류시킵니다(자리가 없으면 대기).
- 이 보상 장수들은 `exclusive:'raid'`로 **영웅전 일반 보상/상점 풀에서 제외**되어 **오직 대장전 보스 격파로만** 획득합니다. (현재 전용 장수는 **7종**: 화웅·원소·사마염·사마의·조조·하후돈·육손. **하후연**은 시작 장수가 되어 전용에서 빠졌으므로, 하후연 보스 보상은 사실상 **중복**이며 결과창에 "이미 보유"로 표시됩니다.)
- 격파한 보스는 `hw_raid_cleared`에 기록되어 목록에 ✔ 표시됩니다(재도전 가능, 보상은 1회).

## 격파 보상 골드

- **첫 격파** 시 영웅전 골드가 `hw_bonus_gold`에 적립되어, 다음 영웅전 지도 진입 시 모험 골드에 반영됩니다.
- 적립 금액은 난이도별 `HW_BOSS[diff].raidGold` — 하(easy) 50 / 중(normal) 100 / 상(hard) 150.
- **재도전 클리어**: 보상 장수를 **아직 수집하지 않았으면 추가**로 영입하고, **이미 보유했으면 보스 단계별 보너스 골드**를 적립합니다 — 보스 순서대로 **+50 ~ +200** 차등(`50,70,90,110,140,160,180,200`).

## 선택 화면 — 장수 · 장비 · 도감

보스 설명 아래에 영웅전 지도 화면과 같은 세 버튼이 있습니다(공유 덱 조회 + 도감). **언락된 보스 카드를 탭하면 보스 정보 팝업**(HP·공격·스킬 2종·보상 장수)이 열립니다(`showBossInfo`).

- **👥 장수**: 공유 출진 덱 **편성** 팝업 — 수집 장수를 탭해 출진 덱(최소 10·최대 30)에 넣고 뺍니다(영웅전과 공유).
- **🗡️ 장비**: 덱 전체에 장착된 무기와 **착용 중인 장수**를 표시.
- **📖 도감**: 영웅전과 동일한 **장수/장비 탭** 도감 카탈로그(공유 `hw_collected_*` 저장 기반).
