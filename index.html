📜 변경 내역 v1.9.1

📅 2026.06.20

  • 대장전 대기실 BGM 분리 · BGM 기본 볼륨 조정 — 대장전 대기실은 영웅전 대기실과 같은 BGM, 전투는 기존 대장전 BGM으로 분리했습니다. 또한 배경음(BGM) 기본 볼륨을 20으로 낮췄습니다(이미 직접 설정한 경우 그 값 유지).
  • 소리 설정 분리 — 효과음/배경음 — 홈 설정의 ‘소리 설정’을 효과음(SFX) 온·오프와 배경음(BGM) 볼륨으로 분리했습니다. BGM은 슬라이더(우측 숫자 0~100)로 조절하며 0이면 자동 오프, 설정값은 기기에 저장됩니다.
  • 화면별 BGM 추가 — 인트로·홈·영웅전 대기실·영웅전 전투·대장전·히어로즈 블러드에 상황별 배경음악을 적용했습니다(크로스페이드 무한 루프, 화면 전환 시 자연스럽게 전환). 전투 승패 정산 시 승리/패배 스팅어가 울리며, 홈 소리 설정으로 BGM도 함께 켜고 끌 수 있습니다.
  • 다국어 — 누락 번역 대거 보강 — 인트로 화면, 게임 데이터 관리, 히어로즈 블러드 규칙·배너, 영웅전 게임 가이드·캠프·상점·주막·보상 팝업, 그리고 영웅전·대장전·히어로즈 블러드의 전투 로그·토스트·정보 팝업 등 동적 문구 전반을 5개 언어(한·영·일·중간체·중번체)로 번역 적용했습니다.
  • 인트로(스플래시) 화면 추가 — 앱 콜드 스타트 시 풀스크린 키비주얼·먹빛/금 오버레이·명조 타이틀(글로우+sweep)·불티 효과와 로딩바를 표시하고, 완료 후 ‘터치하여 시작’으로 홈에 진입합니다(우상단 SKIP으로 즉시 완료). 같은 세션에선 한 번만 표시.
  • 덱 구성 아이콘 통일 — 새 ‘덱 편집’ 아이콘을 덱 구성 페이지 상단 제목과 홈 화면 히어로즈 블러드 카드의 ‘덱 구성’ 버튼(🃏 이모지 대체)에도 동일하게 적용했습니다.
  • 히어로즈 블러드 — 상단 아이콘 교체 — 첫 화면 상단 ‘덱 구성’ 버튼 아이콘을 카드 3장(덱 편집) 아이콘으로, 타이틀 배너의 🎴 이모지를 대전 보드판 아이콘으로 교체했습니다.
  • 획득 알림(토스트) 가독성 개선 — 영웅 영입·보물/유물 획득 등 알림 토스트를 화면 중앙 상단에 크게 띄우고, 금색 테두리·발광·팝 애니메이션과 표시 시간(2초)으로 눈에 잘 띄도록 했습니다.
  • 앱 — 뒤로가기 동작 수정 — 안드로이드에서 뒤로가기를 누르면 어디서든 앱이 백그라운드로 내려가던 문제 수정. 이제 열린 팝업이 있으면 닫고, 없으면 이전 화면으로, 더 갈 곳이 없으면 ‘앱을 종료할까요?’ 확인 후 종료합니다. (@capacitor/app 백버튼 이벤트로 처리)
  • 게임 데이터 관리(파일 전용) — ‘데이터 백업/복구’를 ‘게임 데이터 관리’로 단순화. 메뉴를 내보내기 / 불러오기(파일 전용)로 정리했습니다. 앱(안드로이드)에서도 내보내기가 동작하도록 수정(공유 시트로 파일 저장).
  • 앱 — 뒤로가기 동작 개선 — 안드로이드 하드웨어 뒤로가기 시 이전 화면으로 이동하고, 더 갈 곳이 없으면 ‘앱을 종료할까요?’ 확인 후 종료합니다.
  • 영웅전 — 장수 강화 횟수 제한 — 한 장수당 강화(공격력 +3)는 최대 10회까지만 가능합니다. 강화 팝업에 현재 강화 횟수(n/10)가 표시되고, 10회에 도달한 장수는 선택할 수 없습니다.
  • 영웅전 — 소모품 판매(상시) — 상점에서 보유한 소모품(회복약 등)을 구매가의 50%(11골드)로 언제든 판매할 수 있습니다.
  • 영웅전 — 장비 장착 시 주공 능력치 즉시 반영 — 주공 최대 HP/MP를 올리는 장비를 장착/해제하면 대기실 능력치 바가 바로 갱신됩니다. 최대치 증가분만큼 현재치도 함께 반영(장착/해제 왕복 시 순증가 없음).
  • 히어로즈 블러드 — 상단 금화 즉시 반영 — 승리/패배 정산 골드(영웅전 복귀 시 적용되는 보너스)가 상단 금화에 바로 합산되어 표시되도록 수정.
  • 영웅전 — 보스 능력치·스킬 강화 — 중간보스·메인보스의 공격력 1.5배·HP 1.3배, 보스 공격 스킬 피해 1.5배 상향. 메인보스는 공격 스킬 + (행동 불능 또는 회복) 1종으로 구성되며, 대기실 출진 지역에서 보스를 누르면 나오는 정보에 두 스킬 구성을 표시하도록 갱신.
  • 영웅전 — 유물(오추마·벽사 부적) 변경 — 기본 공격 시 HP 회복 → 스킬 사용 시 HP 회복으로 변경, 수치는 1/2(반올림)로 조정(오추마 2→1, 벽사 부적 3→2). (대장전에도 동일 적용)
  • 영웅전 — 상점 ‘영웅 강화’ 개편 — ‘무기 강화’를 ‘영웅 강화’로 이름 변경, 약 40% 확률로 진열. 대상이 무작위가 아니라 장수를 직접 선택(팝업)하도록 변경. 한 번 사용하면 새로고침해도 다음 스테이지 클리어까지 비활성 유지.
  • 영웅전 — 캠프 ‘장수 강화’ 선택 팝업 — 캠프의 장수 강화도 인라인 목록 대신 장수 선택 팝업으로 변경.
  • 데이터 백업 / 복구 추가 — 홈 설정 → 데이터 백업 / 복구에서 모든 진행·컬렉션·설정을 백업 코드로 내보내고(복사 또는 파일 저장), 코드를 붙여넣어(또는 파일 불러오기) 복구할 수 있습니다. 앱 재설치 전 백업해 두면 진행을 잃지 않습니다.
  • 도감 탭 표시 수정 — 도감이 탭 구분 없이 장수·장비·유물이 한꺼번에 나오던 문제 수정(패널의 display:flex가 숨김을 덮어쓰던 버그). 이제 선택한 탭만 보입니다.
  • 영웅전 — 보물상자(무기) 보상 팝업화 — 출진 5·10회째 나오던 보물상자(무기/보패) 보상을 전용 페이지에서 대기실 팝업으로 변경(유물·중간보스 보상과 동일한 흐름).
  • 영웅전 — 캠프 시점 이동(5→9스테이지) — 캠프 정비를 9스테이지 클리어 후에 나오도록 옮겼습니다(기존 5스테이지).
  • 히어로즈 블러드 — 상단 연승 표시 — 헤더 골드 칩 옆에 🔥 연승 칩을 추가했습니다. 0연승은 흐리게, 1연승부터 적색으로 강조됩니다. 실제 연승값과 연동되어 판을 넘어 유지됩니다.
  • 히어로즈 블러드 — AI 전용 카드 하향(밸런스) — 역전의 명장 무력 8→4, 책략가 ‘같은 줄 적 -2’→‘앞 칸 적 -2’·무력 5→3, 맹장 무력 4→2. 상향됐던 상대 난이도를 다소 완화했습니다(시뮬레이션 기준 AI 승률 3×5 ≈ 69% · 4×5 ≈ 76% · 5×5 ≈ 88%).
  • 히어로즈 블러드 — 카드 정보 팝업의 강화 영역 격자 표시 수정 — 보드에 놓인 장수를 탭하면 나오는 정보 팝업에서 강화 영역 격자(3×3)가 보이지 않던 문제 수정. 가운데 금색=카드 위치, 녹색=강화(점령 확장) 칸으로 표시됩니다.
  • 영웅전 — 대기실 진입 시 진행 위치로 자동 스크롤 — 대기실에 들어가면 출진 지역 목록이 현재 진행 중인 스테이지로 자동 스크롤됩니다.
  • 히어로즈 블러드 — 상대 난이도 대폭 상향 — AI 전용 강카드 4장(낮은 등급+강한 효과로 항상 전개 가능)과 낮은 곡선의 덱을 도입해 AI 승률을 크게 끌어올렸습니다(시뮬레이션 기준 약 55% → 80~94%). AI 전용 카드는 덱 구성·도감에 노출되지 않습니다.
  • 히어로즈 블러드 — 상대 카드 영역 정보 표시 — 보드에 놓인 상대 카드를 탭하면 정보 팝업에 강화 영역(점령 확장 패턴)이 상대 시점 기준으로 함께 표시됩니다.
  • 영웅전/대장전 — 전투 화면 개선 묶음 — ① 아군 카드의 중독·상태이상 표시를 카드 오른쪽 상단으로 옮겨 공격력 배지에 가려지지 않게 했습니다. ② 전투 화면 상단 왼쪽에 이전(대기실로) 버튼 추가. ③ 스테이지 보스·대장전 보스 치명타 확률 최소 15% 적용. ④ 턴 종료 시 손패가 비어도 손패 영역 높이를 유지(‘카드 없음’ 표시로 높이가 출렁이던 문제 해결). ⑤ 영웅전 중간보스 승리 보상을 전용 페이지 대신 대기실 팝업으로 변경.
  • 영웅전 — 보물 발견 확률 복구 — 테스트용으로 100%였던 보물 발견 이벤트 출현 확률을 정상값 10%로 되돌렸습니다.
  • 히어로즈 블러드 — AI 난이도 복구(점령 규칙 대응) — 점령(빼앗기) 규칙 도입으로 선공(플레이어)이 유리해져 약해진 AI를 보강했습니다. 영역 가치 가중치 ↑, 상대의 반격 경계 ↑, 특히 상대가 내 영역을 빼앗는 강수를 더 경계하도록 조정해 큰 보드(특히 5×5)에서 떨어졌던 승률을 회복했습니다.
  • 히어로즈 블러드 — 상대 카드 강화 패턴 방향 표시 보정 — 보드에서 상대 카드를 탭해 볼 때 강화 패턴이 내 시점(전방=위)으로만 표시되던 것을, 상대 시점(전방=아래)으로 그려 실제 확장 방향과 일치시켰습니다. (내 카드 표시는 그대로)
  • 히어로즈 블러드 — 카드 스킬 가독성 개선 — 덱 구성 카드의 높이를 키워 스킬 설명이 우하단 강화 격자 아이콘에 가려지던 문제 해소, 손패 카드의 스킬 이름 위치를 위로 올려 격자 아이콘과 겹치지 않게 했습니다.
  • 히어로즈 블러드 — 점령(빼앗기) 규칙 추가 — 카드의 강화 패턴이 상대가 점령한 빈 칸(카드 없음)에 닿으면 그 칸을 빼앗아 내 폰 1로 초기화합니다. 카드가 놓인 칸은 빼앗을 수 없습니다. AI(상대)에게도 동일하게 적용됩니다.
  • 보스 격파 마지막 대사 — 영웅전 적장·대장전 레이드 보스를 격파하면 그 보스의 마지막 대사가 나온 뒤 보상으로 넘어갑니다(대사 설정 ON일 때).
  • 영웅전 유물 보상 — 대기실 팝업화 — 적장 격파 후 나오던 유물 선택 화면을 대기실 팝업으로 변경(보물 발견 팝업과 동일한 흐름).

  • 팝업 높이 보정 — 도감 등 모달 팝업 높이를 줄이고 내부 목록 창 높이를 뷰포트에 맞춰 조절. 내용이 길면 모달이 스크롤(auto)되어 닫기 버튼까지 닿게(모바일 동적 뷰포트 대응).
  • 일반 출진 보상 정리 — 일반 출진(중간보스·적장 제외)에서 주던 장수 영입 보상을 골드 보상으로 변경. 장수 영입은 중간보스 격파·주막·특수 경로로만.
  • 유물 15종으로 확장 — 독항아리(전투 시작 적 중독)·옥천수(승리 시 MP+10)·현철 갑주(최대 HP+20)·예기(치명타+8%) 등 신규 유물 추가, 군량미 승리 회복 +6 → +20 상향. 영웅전·대장전 동시 적용.
  • 천자의 밀서(유물) — 히어로즈 블러드 10연승 보상(1회), 골드 보상 획득량 +20%.

⚔️ 삼국 영웅전 — 로그라이크 카드 전투

  • 당신은 주공입니다. 8개 역사 전역을 차례로 평정해 천하통일이 목표.
  • 전투는 가운데 장수 카드로 적을 공격하고, 스킬은 주공 MP를 씁니다. 턴에 쓰지 않은 가운데 카드는 버려집니다.
  • 적은 주공 HP를 노립니다 — 0이 되면 패배. HP·MP는 모험 내내 이어집니다.
  • 지도에서 🀄 진형(출진 덱 10~30장) · 👥 장수(정보·무기 장착) · 🏪 상점(무기·소모품) · 🏮 주막(장수 영입)으로 부대를 키우세요.
  • 5·10 출진은 중간보스, 11출진은 적장. 지도의 ⚜·👑 칸을 탭하면 적 정보를 봅니다.

🀄 히어로즈 블러드 — 보드 카드 대전

  • 덱 구성에서 카드 15장과 판 크기(3×5·4×5)를 정합니다.
  • 카드는 칸의 폰 레벨 이하 등급만 놓을 수 있고, 놓으면 강화 패턴대로 빈 칸에 폰을 부여해 영역을 넓힙니다(최대 3).
  • 배치한 카드의 무력 총합이 높은 쪽이 승리. 내가 패스 2번이면 정산합니다.
  • 결과는 영웅전 골드로 정산되고, 3연승 시 제갈량을 얻습니다(연승 보너스 골드도).

👹 삼국 대장전 — 레이드 보스전

  • 영웅전 덱을 공유해 거대 HP의 적장(레이드 보스)에 도전합니다.
  • 보스는 2종 스킬과 MP를 가지며 혼란·매혹엔 면역(대장전에선 그 스킬이 다른 스킬로 교체).
  • 격파하면 그 적장을 전용 장수로 영입 — 오직 대장전에서만 얻습니다. 재도전 시 골드 보상.
  • 전투 중 영웅전과 공유하는 소모품을 쓸 수 있습니다.
