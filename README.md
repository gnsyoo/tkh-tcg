# 삼국지 TCG 게임 — 웹 목업

**삼국 영웅전**(Heroes Wanted 스타일 로그라이크 TCG)과, 특정 보스/지역에서 플레이되는
**히어로즈 블러드**(파이널판타지7 리버스의 Queen's Blood 스타일 보드 카드 게임)를 웹으로 만든 목업입니다.
삼국지 장수 테마로 리스킨되어 있습니다(장수별 스탯은 추후 세팅 예정).
구현 가능 여부와 게임성 확인을 목적으로 합니다.

> 최종 빌드는 Unity로 진행하되, 빠른 확인을 위해 먼저 웹으로 프로토타이핑했습니다.

## 구현 언어 (추천)

**HTML + CSS + Vanilla JavaScript** (프레임워크·빌드 도구 없음)

추천 이유:
- **즉시 실행 / 무설치** — 파일을 브라우저로 열기만 하면 PC·모바일 어디서나 동작하며, 빌드 단계가 없습니다.
- **모바일 지원** — 반응형 레이아웃과 탭(터치) 기반 조작으로 휴대폰에서 그대로 플레이됩니다.
- **의존성 0** — 외부 라이브러리/이미지 서버 없이 동작(아트는 이모지 + CSS 그라데이션 사용)하므로 오프라인에서도 안정적입니다.
- **이식성** — 카드 데이터와 규칙 로직이 순수 함수로 분리되어 있어 추후 Unity(C#)로 규칙을 옮기기 쉽습니다.

## 실행 방법

가장 간단한 방법은 `index.html`을 브라우저로 직접 여는 것입니다.

로컬 서버로 실행하려면(권장):

```bash
# 프로젝트 폴더에서
python3 -m http.server 8000
# 브라우저에서 http://localhost:8000 접속
```

모바일에서 플레이하려면 PC와 같은 와이파이에 연결한 뒤 `http://<PC-IP>:8000` 으로 접속하세요.

## 화면 구성

- **첫 페이지(`index.html`)** — 난이도(상/중/하)를 고르고 두 게임 중 하나를 선택해 진입.
- **삼국 영웅전(`heroes.html`)** — 로그라이크 파티 전투.
- **삼국 대장전(`daejang.html`)** — 영웅전 덱을 공유하는 레이드 보스전.
- **히어로즈 블러드(`queensblood.html`)** — 3~4줄 × 5칸 보드 카드 배치 게임.
- 두 게임 모두 **메뉴는 우측 상단 플로팅 버튼**(누르면 펼쳐짐)으로 통합되어 있습니다.

난이도는 첫 페이지에서 선택하며, 선택값은 URL 파라미터(`?diff=`)와 `localStorage`로 전달됩니다.

## 게임 1 — 삼국 영웅전 (로그라이크 파티 전투)

- **주공(나)**: 플레이어는 **주공**으로, 시작 **HP 200 · MP 50**을 가집니다. 적군은 개별 장수가 아니라 **주공의 HP를 공격**하며, 주공이 쓰러지면 패배합니다. **HP·MP 모두 모험 내내 유지**(소모전)됩니다 — HP는 승리 시 일부·전역 평정 시 완전 회복, **MP는 전투 시작 시 10% 회복**·저잣거리 두강주(MP+20)로 보충. **스킬 사용 시 MP 3~5 소모**.
- **치명타**: 기본 공격이 기본 1% 확률로 **공격력 2배** 피해. **적도 1% 확률로 치명타**. 치명타 확률을 올리는 무기(청강검/의천검/자웅일대검)도 있습니다.
- **전역(스테이지)**: **8개 역사 전투**(반동탁 연합군 191년 → 관도대전 → 장판파 → 적벽대전 → 정군산 → 이릉대전 → 제갈량의 북벌 → 위·촉·오 멸망)를 차례로 평정. **뒤 스테이지일수록 난이도가 상승**하고, 각 스테이지에는 그 시대의 적장(화웅·원소·조조·육손·사마의·사마염 등)이 등장.
- **파티·무기**: 시작 장수 4명. 전투 보상으로 장수를 영입(최대 30명, 가득 차면 교체, **중복 수집 불가**). **무기 장착 수는 등급별**(C=0·R=1·SR=2·SSR=3) — 공격력+/2회 연속/독/치명타/회피(명마)·주공 HP·MP 증가 등 29종. **출진 5·10회 뒤 보물상자**·**저잣거리(상점)**에서 획득.
- **메뉴 → 데이터 초기화**: 우측 상단 메뉴의 **🗑️ 데이터 초기화**로 삼국 영웅전·삼국 대장전의 모든 진행과 장수/장비 컬렉션·모드 해금 기록을 한 번에 삭제할 수 있습니다(확인 후 적용).
- **카드 전투**: 하단이 3구역(왼쪽=뽑을 더미 / 가운데=공격·방어 덱 3장 / 오른쪽=사용한 더미).
  - 가운데 카드를 **쓰면 공격**(기본공격/스킬+대상), **턴 종료 시 남은 카드는 방어**해 주공의 블록을 올리고 사용한 더미로 이동.
  - 뽑을 더미가 비면 사용한 더미를 랜덤 셔플해 보충. 적은 다음 행동을 머리 위에 **예고(intent)**.
  - **매혹**: 초선·소교·대교의 스킬은 적 1명을 1턴 매혹시켜 행동 불가로 만듭니다.
  - 베기·타격·치명타·파티클·화면 흔들림·배너 등 전투 연출 포함.
- **53종의 삼국지 장수**(전사/수호/책략/궁수/치유/기습/무희/군주 등). 등급 **C·R·SR·SSR**, SSR은 능력치가 높고 무기 3개 장착(사마의는 총사령관급 OP). 적장이던 **화웅·원소·사마염**도 일반 장수로 편입되어 영입/구매·대장전 보상으로 얻을 수 있습니다.
- **전용(획득 경로 제한) 장수**: **제갈량**만 히어로즈 블러드 **3연승** 시 획득하는 전용 장수입니다. 대장전 보스를 격파하면 **해당 적장(화웅·원소·하후돈·조조·하후연·육손·사마의·사마염)을 일반 장수로 영입**합니다(별도의 전용 변종이 아니라 그대로 그 장수).
- **천하통일 → 엔딩 크레딧**: 마지막 적장(사마염)까지 격파하면 평정 전역·장수 명단·유물·난이도가 흐르는 엔딩 크레딧이 재생됩니다.
- **두 게임 연동**: 히어로즈 블러드 승리 시 포인트만큼 영웅전 골드 **추가**, 패배 시 포인트만큼 **차감**.

## 게임 2 — 히어로즈 블러드 (보드 카드 게임)

- **진입**: 들어가면 **덱 구성 화면부터** 표시 — 카드 15장과 **판 크기(3×5 · 4×5)**를 고르고 시작합니다.
- **보드**: 가로 3~4줄 × 세로 5칸(선택). 당신은 왼쪽 끝 열을, 상대는 오른쪽 끝 열을 폰 1레벨로 시작.
- **카드 배치**: 자신이 점령한 칸에, 그 칸의 **폰 레벨(♟️) 이상의 등급(핍)**을 가진 카드만 배치 가능. (예: 레벨 2 칸 → 등급 2·3·4·5 카드)
- **확장**: 카드를 놓으면 카드의 강화 패턴에 따라 빈 칸에 폰을 +1 부여(최대 3)하며 영역을 넓힘.
- **능력**: 일부 카드는 아군 파워 증가/적 파워 감소 등 지속 효과 보유.
- **승패**: 보드에 **배치한 모든 카드의 무력(파워) 총합**이 높은 쪽이 승리.
- **종료**: **당신이 패스를 2번** 하면 종료 여부를 물어본 뒤, 확인하면 정산합니다(놓을 카드가 없으면 자동 패스). 상대의 패스만으로는 끝나지 않습니다.
- **골드 정산**: 종료 시 내 무력 총합(포인트)만큼 영웅전 골드를 승리=가산 / 패배=차감합니다.
- **장수 카드 44종**(요구 등급·무력·강화 패턴·능력이 모두 다른) 수록. 덱 빌더로 15장 덱 구성·저장.
- AI 상대: 난이도에 따라 무작위(하) → 탐욕적 평가(중) → 1수 앞 예측(상)으로 강해집니다.
- **연승 보상**: 3연승 달성 시 영웅전 전용 SSR **제갈량**을 획득합니다.

## 게임 3 — 삼국 대장전 (레이드 보스전)

- 삼국 영웅전의 **덱(파티)을 그대로 공유**해, 8명의 적장을 **메인 스테이지의 약 12배 HP**를 가진 레이드 보스로 토벌합니다.
- 전투 규칙은 영웅전과 동일(주공 HP/MP·3구역 카드·공격/방어/스킬·치명타·매혹).
- **보스를 격파하면 그 적장을 일반 장수로 영입**(영웅전 파티에 합류). 별도의 전용 변종(예: "맹장 화웅")이 아니라 **그 장수 본인(화웅 등)** 을 그대로 얻습니다 — 화웅·원소·사마염은 이 보상으로 추가된 일반 장수이고, 하후돈·조조·하후연·육손·사마의는 기존 로스터의 그 장수입니다.
- 보스는 **앞 보스를 격파해야 다음이 해금**되며 진행 상태가 저장됩니다.

## 난이도(상/중/하)

| 게임 | 하(Easy) | 중(Normal) | 상(Hard) |

| 히어로즈 블러드 | 무작위 AI | 탐욕적 AI | 1수 예측 AI·강한 덱 |
