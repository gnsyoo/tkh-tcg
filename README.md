# 삼국지 TCG 게임 — 웹 목업

**삼국지 영웅모집**(Heroes Wanted 스타일 로그라이크 TCG)과, 특정 보스/지역에서 플레이되는
**히어로즈 블러드**(파이널판타지7 리버스의 Queen's Blood 스타일 보드 카드 게임)를 웹으로 만든 목업입니다.
삼국지 장수 테마로 리스킨되어 있습니다(장수별 스탯은 추후 세팅 예정).
구현 가능 여부와 게임성 확인을 목적으로 합니다.

> 최종 빌드는 Unity로 진행하되, 빠른 확인을 위해 먼저 웹으로 프로토타이핑했습니다.

## 구현 언어 (추천)

**HTML + CSS + Vanilla JavaScript** (프레임워크·빌드 도구 없음)

추천 이유:
- **즉시 실행 / 무설치** — 파일을 브라우저로 열기만 하면 PC·모바일 어디서나 동작하며, 빌드 단계가 없습니다.
- **모바일 지원** — 반응형 레이아웃과 탭(터치) 기반 조작으로 휴대폰에서 그대로 플레이됩니다.
- **의존성 0** — 외부 라이브러리/이미지 서버 없이 동작(아트는 이모지 + CSS 그라데이션 사용)하므로 오프라인에서도 안정적입니다.
- **이식성** — 카드 데이터와 규칙 로직이 순수 함수로 분리되어 있어 추후 Unity(C#)로 규칙을 옮기기 쉽습니다.

## 실행 방법

가장 간단한 방법은 `index.html`을 브라우저로 직접 여는 것입니다.

로컬 서버로 실행하려면(권장):

```bash
# 프로젝트 폴더에서
python3 -m http.server 8000
# 브라우저에서 http://localhost:8000 접속
```

모바일에서 플레이하려면 PC와 같은 와이파이에 연결한 뒤 `http://<PC-IP>:8000` 으로 접속하세요.

## 화면 구성

- **첫 페이지(`index.html`)** — 난이도(상/중/하)를 고르고 두 게임 중 하나를 선택해 진입.
- **삼국지 영웅모집(`heroes.html`)** — 로그라이크 파티 전투.
- **히어로즈 블러드(`queensblood.html`)** — 3×5 보드 카드 배치 게임.

난이도는 첫 페이지에서 선택하며, 선택값은 URL 파라미터(`?diff=`)와 `localStorage`로 전달됩니다.

## 게임 1 — 삼국지 영웅모집 (로그라이크 파티 전투)

- **전역(스테이지)**: **8개 역사 전투**(반동탁 연합군 191년 → 관도대전 → 장판파 → 적벽대전 → 정군산 → 이릉대전 → 제갈량의 북벌 → 위·촉·오 멸망)를 차례로 평정. **뒤 스테이지일수록 난이도가 상승**하고, 각 스테이지에는 그 시대의 적장(화웅·원소·조조·육손·사마의·사마염 등)이 등장.
- **파티**: 시작 장수 4명. 전투 보상으로 장수를 영입(최대 6명, 가득 차면 교체). 출진 전 **저잣거리(상점)** 이용 가능. 각 스테이지 시작 시 파티가 완전 회복.
- **턴제 전투**:
  - 매 라운드 **에너지(⚡)** 를 받고, 장수별로 **기본 공격(무료)** 또는 **스킬(에너지 소모)** 을 한 번씩 사용.
  - 적은 다음 행동(공격/전체공격)을 머리 위에 **예고(intent)** 합니다.
  - 장수 탭 → 행동 선택 → 대상 선택. 베기·타격·파티클·화면 흔들림·배너 등 전투 연출 포함.
- **38명의 삼국지 장수**(전사/수호/책략/궁수/치유/기습 등)와 다양한 스킬(강타·화공(광역)·연속·치유·방어막·고무).

## 게임 2 — 히어로즈 블러드 (보드 카드 게임)

- **보드**: 가로 3줄 × 세로 5칸. 당신은 왼쪽 끝 열을, 상대는 오른쪽 끝 열을 폰 1레벨로 시작.
- **카드 배치**: 자신이 점령한 칸의 **폰 레벨(♟️)** 이 카드의 **요구 등급(핍)** 이상일 때만 배치 가능.
- **확장**: 카드를 놓으면 카드의 강화 패턴에 따라 빈 칸에 폰을 +1 부여(최대 3)하며 영역을 넓힘.
- **능력**: 일부 카드는 아군 파워 증가/적 파워 감소 등 지속 효과 보유.
- **승패**: 보드에 **배치한 모든 카드의 무력(파워) 총합**이 높은 쪽이 승리.
- **종료**: **양쪽이 연달아 패스**하면 종료 여부를 확인한 뒤 정산합니다(놓을 카드가 없으면 자동 패스).
- **장수 카드 44종**(요구 등급·무력·강화 패턴·능력이 모두 다른) 수록. 덱 빌더로 15장 덱 구성·저장.
- AI 상대: 난이도에 따라 무작위(하) → 탐욕적 평가(중) → 1수 앞 예측(상)으로 강해집니다.

## 난이도(상/중/하)

| 게임 | 하(Easy) | 중(Normal) | 상(Hard) |
|------|----------|------------|----------|
| 삼국지 영웅모집 | 적 약화·골드↑·단순 AI | 기본 | 적 강화·골드↓·영리한 AI(최저 HP 우선 타격) |
| 히어로즈 블러드 | 무작위 AI | 탐욕적 AI | 1수 예측 AI·강한 덱 |
